# Design System Aurora Tech

Sistema de Design oficial da **Aurora Tech** — moderno, acessível e alinhado com nossa identidade de tecnologia e IA.

> Versão 1.0 • Abril 2026

## Sobre

Este repositório contém a documentação completa do Design System da Aurora Tech, incluindo paleta de cores, tipografia, componentes UI, espaçamento, ícones e guias de aplicação.

A documentação completa está em [Sistema_Design_Aurora_Tech.md](./Sistema_Design_Aurora_Tech.md).

## Sumário

1. **Paleta de Cores** — Cyan primário, preto, branco e cinzas, com cores semânticas (success, error, warning, info).
2. **Tipografia** — Montserrat (display), Inter (body), Fira Code (código).
3. **Componentes UI** — Botões, cards, inputs, badges/tags.
4. **Espaçamento & Layout** — Sistema de 8px, grid responsivo (12/8/4 colunas), border radius padronizado.
5. **Sistema de Ícones** — Stroke 1.5px, base 24px, estilo minimalista geométrico.
6. **Guia de Aplicação** — Combinações recomendadas, boas práticas e o que evitar.

## Princípios

- **Dark-first** — Tema principal com fundo preto e destaques cyan.
- **Acessibilidade** — Contraste mínimo 4.5:1 em todo conteúdo.
- **Consistência** — Tokens e padrões reutilizáveis em toda a plataforma.
- **Moderno** — Linhas limpas, efeitos neon sutis, tipografia legível.

## Paleta resumida

| Token | Hex | Uso principal |
|---|---|---|
| Cyan | `#1234AB` | Primária, CTAs, destaques |
| Preto | `#000000` | Fundo principal |
| Branco | `#FFFFFF` | Texto sobre fundo escuro |
| Cinza Claro | `#999999` | Texto secundário |
| Cinza Escuro | `#333333` | Backgrounds alternativos |
