# 1. Paleta de Cores

Nossa paleta é composta por cores que transmitem inovação, tecnologia e confiança. O **Azul** é nossa cor primária, representando a vanguarda tecnológica.

| Nome | Código | Uso | Preview |
|---|---|---|---|
| **Azul (Primária)** | `#1234AB` | Botões, links, destaque | 🟦 |
| Preto (Fundo) | `#000000` | Fundo principal, texto | ⬛ |
| Branco (Texto) | `#FFFFFF` | Texto sobre fundo escuro | ⬜ |
| Cinza Claro | `#999999` | Texto secundário | ▫️ |
| Cinza Escuro | `#333333` | Backgrounds alternativos | ◾ |
| Verde Sucesso | `#00D451` | Status positivo, confirmação | 🟩 |
| Vermelho Erro | `#FF3333` | Erros, aviso crítico | 🟥 |
| Amarelo Aviso | `#FFB800` | Avisos, atenção | 🟨 |

## Psicologia das Cores

- **Azul:** Representa inovação, tecnologia, confiança e futuro. Principal na marca Aurora Tech.
- **Preto:** Sofisticação, modernidade e foco. Cria contraste com elementos azuis.
- **Branco:** Limpeza, clareza e legibilidade. Essencial para acessibilidade.
- **Cinza:** Neutralidade. Texto secundário e fundos alternativos.

# 2. Tipografia

A tipografia da Aurora Tech foi escolhida para ser moderna, legível e profissional.

## Famílias de Fonte

| Categoria | Fonte | Uso |
|---|---|---|
| Display/Títulos | Montserrat Bold | Títulos principais, headings, destaques |
| Body/Corpo | Inter Regular | Textos corridos, descrições, conteúdo |
| Código | Fira Code | Código, snippets técnicos |

## Tamanhos de Fonte

| Elemento | Tamanho | Peso | Contexto |
|---|---|---|---|
| H1 - Título Principal | 28pt | Bold | Títulos de página |
| H2 - Subtítulo | 16pt | Bold | Seções principais |
| H3 - Subtítulo Menor | 14pt | Semibold | Subsseções |
| Body Regular | 12pt | Regular | Texto do corpo |
| Body Small | 11pt | Regular | Texto secundário |
| Caption | 10pt | Regular | Legendas, notas rodapé |

## Espaçamento Vertical

- **Headings: 1.2x (tight)** • Impacto visual maior
- **Body: 1.5x (relaxed)** • Legibilidade otimizada
- **Captions: 1.3x** • Compacto mas legível

# 3. Componentes UI

Componentes reutilizáveis que mantêm consistência visual em toda a plataforma.

## Botões

- **Primary Button:** Fundo Azul, texto preto — Ações principais, CTAs
- **Secondary Button:** Borda Azul, fundo transparente — Ações secundárias
- **Disabled Button:** Cinza 333, opacidade 50% — Estado inativo

## Cards

- Bordas: Azul 1px (opcional)
- Fundo: Preto/Cinza 333
- Padding: 20px
- Border Radius: 8px
- Sombra: Azul brilhante, baixa opacidade (efeito neon)

## Campos de Input

- Borda padrão: Cinza 999
- Borda ativa (focus): Azul com sombra
- Fundo: Preto ou Cinza 333
- Texto: Branco
- Border Radius: 4px

## Badges/Tags

- **Success:** Fundo Verde, texto branco
- **Error:** Fundo Vermelho, texto branco
- **Warning:** Fundo Amarelo, texto preto
- **Info:** Fundo Azul, texto preto

# 4. Espaçamento & Layout

## Sistema de Espaçamento

Usamos um sistema baseado em 8px como unidade fundamental:

| Nível | Valor | Uso |
|---|---|---|
| xs | 4px | Espaço mínimo entre elementos |
| sm | 8px | Espaçamento padrão interno |
| md | 16px | Espaçamento entre seções |
| lg | 24px | Espaçamento maior entre blocos |
| xl | 32px | Espaçamento extra, entre páginas |
| 2xl | 48px | Espaço de respiração máximo |

## Grid & Responsividade

- **Desktop:** 12-coluna grid com 20px gap
- **Tablet:** 8-coluna grid com 16px gap
- **Mobile:** 4-coluna grid com 12px gap

## Bordas (Border Radius)

- **Subtle (4px):** Inputs, pequenos elementos
- **Standard (8px):** Cards, botões
- **Rounded (12px):** Containers, panels
- **Full (50%):** Avatares, badges

# 5. Sistema de Ícones

Os ícones da Aurora Tech seguem um estilo moderno e minimalista.

## Características

- **Stroke:** 1.5px para melhor legibilidade
- **Tamanho base:** 24px (escalável para 16px, 32px, 48px)
- **Cor padrão:** Branco ou Azul (conforme contexto)
- **Espaçamento interno:** Margem de 2px dentro da caixa
- **Estilo:** Linhas limpas, sem preenchimento, geométrico

## Categorias de Ícones

- **Navigation:** Menu, Home, Voltar, Próximo, Fechar
- **UI Controls:** Settings, Filter, Search, Upload, Download
- **Communication:** Chat, Mail, Bell, Share
- **Status:** Check, Close, Warning, Info, Help
- **Media:** Image, Video, Play, Pause, Volume

# 6. Guia de Aplicação

## Combinações de Cor Recomendadas

- **Dark Theme (Principal):** Fundo Preto + Texto Branco + Destaque Azul
- **Light Theme (Alternativa):** Fundo Branco + Texto Preto + Destaque Azul
- **Neon Effect:** Fundo Preto + Bordas Azul + Sombras Azul brilhantes

## Boas Práticas

- Mantenha contraste mínimo 4.5:1 para acessibilidade
- Use Azul com moderação — é cor de destaque, não fundo
- Respeite o espaçamento entre elementos
- Priorize tipografia limpa e legível
- Teste componentes em diferentes resoluções
- Use sombras sutis para profundidade, não dramaticidade
- Mantenha consistência no tamanho de ícones e botões

## O Que Evitar

- Não misture Azul com outras cores vibrantes
- Não use texto branco sobre fundo Azul (contraste baixo)
- Não redimensione ícones arbitrariamente
- Não ignore o espaçamento definido
- Não use mais de 3 cores principais por página
- Não aplique efeitos de sombra excessivos
