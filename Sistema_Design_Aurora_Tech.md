# 1. Paleta de Cores

Nossa paleta é composta por cores que transmitem inovação, tecnologia e confiança. O cyan é nossa cor primária, representando a vanguarda tecnológica.

## Psicologia das Cores

- Cyan: Representa inovação, tecnologia, confiança e futuro. Principal na marca Aurora Tech.

- Preto: Sofisticação, modernidade e foco. Cria contraste com elementos cyan.

- Branco: Limpeza, clareza e legibilidade. Essencial para acessibilidade.

- Cinza: Neutralidade. Texto secundário e fundos alternativos.

# 2. Tipografia

A tipografia da Aurora Tech foi escolhida para ser moderna, legível e profissional.

## Tamanhos de Fonte

## Espaçamento Vertical

# 3. Componentes UI

Componentes reutilizáveis que mantêm consistência visual em toda a plataforma.

## Botões

- Primary Button: Fundo Cyan, texto preto — Ações principais, CTAs

- Secondary Button: Borda Cyan, fundo transparente — Ações secundárias

- Disabled Button: Cinza 333, opacidade 50% — Estado inativo

## Cards

- Bordas: Cyan 1px (opcional)
Fundo: Preto/Cinza 333
Padding: 20px
Border Radius: 8px
Sombra: Cyan brilhante, baixa opacidade (efeito neon)

## Campos de Input

- Borda padrão: Cinza 999
Borda ativa (focus): Cyan com sombra
Fundo: Preto ou Cinza 333
Texto: Branco
Border Radius: 4px

## Badges/Tags

- Success: Fundo Verde, texto branco

- Error: Fundo Vermelho, texto branco

- Warning: Fundo Amarelo, texto preto

- Info: Fundo Cyan, texto preto

# 4. Espaçamento & Layout

## Sistema de Espaçamento

Usamos um sistema baseado em 8px como unidade fundamental:

## Grid & Responsividade

- Desktop: 12-coluna grid com 20px gap
Tablet: 8-coluna grid com 16px gap
Mobile: 4-coluna grid com 12px gap

## Bordas (Border Radius)

- Subtle (4px): Inputs, pequenos elementos

- Standard (8px): Cards, botões

- Rounded (12px): Containers, panels

- Full (50%): Avatares, badges

# 5. Sistema de Ícones

Os ícones da Aurora Tech seguem um estilo moderno e minimalista.

## Características

- Stroke: 1.5px para melhor legibilidade

- Tamanho base: 24px (escalável para 16px, 32px, 48px)

- Cor padrão: Branco ou Cyan (conforme contexto)

- Espaçamento interno: Margem de 2px dentro da caixa

- Estilo: Linhas limpas, sem preenchimento, geométrico

## Categorias de Ícones

- Navigation: Menu, Home, Voltar, Próximo, Fechar

- UI Controls: Settings, Filter, Search, Upload, Download

- Communication: Chat, Mail, Bell, Share

- Status: Check, Close, Warning, Info, Help

- Media: Image, Video, Play, Pause, Volume

# 6. Guia de Aplicação

## Combinações de Cor Recomendadas

- Dark Theme (Principal): Fundo Preto + Texto Branco + Destaque Cyan

- Light Theme (Alternativa): Fundo Branco + Texto Preto + Destaque Cyan

- Neon Effect: Fundo Preto + Bordas Cyan + Sombras Cyan brilhantes

## Boas Práticas

- Mantenha contraste mínimo 4.5:1 para acessibilidade

- Use Cyan com moderação — é cor de destaque, não fundo

- Respeite o espaçamento entre elementos

- Priorize tipografia limpa e legível

- Teste componentes em diferentes resoluções

- Use sombras sutis para profundidade, não dramaticidade

- Mantenha consistência no tamanho de ícones e botões

## O Que Evitar

- Não misture Cyan com outras cores vibrantes

- Não use texto branco sobre fundo Cyan (contraste baixo)

- Não redimensione ícones arbitrariamente

- Não ignore o espaçamento definido

- Não use mais de 3 cores principais por página

- Não aplique efeitos de sombra excessivos
